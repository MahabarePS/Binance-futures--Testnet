# Binance-futures--Testnet
A Python-based command-line application for placing and managing Binance USDT-M Futures Testnet orders. The project demonstrates clean software architecture, input validation, API integration, logging, and exception handling.
# Project Overview
This project provides a reusable trading client that interacts with the Binance Futures Testnet environment.

The application supports:

- Market Orders
- Limit Orders
- BUY and SELL sides
- Input validation
- Structured logging
- Error handling
- Command Line Interface (CLI)

The primary objective is to demonstrate software engineering best practices rather than trading strategy development.

# Separation of Concerns
Validation, business logic, and API communication are separated into independent modules.

Benefits:

- Easier testing
- Better maintainability
- Improved scalability
- Reduced coupling
